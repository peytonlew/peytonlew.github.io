Super Bowl
Analysis

This project analyzes past super bowl data.

This project was done through DataCamp within the Data Science career track. This project uses basic and intermediate Python knowledge to analyze data collected about past Super Bowls and the tv ratings going along with it.

Here, you will see comparisons between Super Bowl games, the range of how close games can be, and how those differences impact the ratings of TV viewers.
